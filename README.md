# HR-Analytics-Dashboard-Using-Tableau

# Project Overview

This project presents an interactive HR Analytics Dashboard built using Tableau to analyze employee data and uncover insights related to attrition, demographics, and job satisfaction.

The goal is to help organizations make data-driven HR decisions by identifying patterns and trends in workforce data.

# Key Features
 Attrition Analysis (Overall & Department-wise)
 Employee Distribution by Age Group
 Gender-based Attrition Comparison
 Education Field-wise Attrition Insights
 Job Satisfaction Rating Analysis
 KPI Metrics (Employee Count, Attrition Rate, Active Employees, Avg Age)
 Dashboard Preview



# Key Insights
Highest attrition observed in specific departments
Age group 30–35 has the highest number of employees
Attrition rate differs significantly across gender
Employees from Life Sciences & Medical fields show higher attrition
Job satisfaction directly impacts employee retention

# Tools & Technologies Used
Tableau (Dashboard & Visualization)
Excel / CSV (Dataset)
Data Cleaning & Transformation
